# Personality Layer

The codebase includes a light personality layer rather than a full narrative route system.

## Implemented Elements

- `FlavorTextManager` floating text lines on entity interactions
- Mercy heart tracking from near misses
- Animal-specific reactions such as fox and cat dialogue
- Ghost replay of a best run
- Rest / game-over presentation and music shift

## Not In This Repo

- Branching story routes
- Dialogue trees
- Save-slot narrative flags
- Complex pacifist/genocide route logic

The older documents overstated this area. The current repo supports flavor and feedback, not a full narrative RPG system.
